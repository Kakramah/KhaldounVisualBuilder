# Template 01 — Manifesto

## متى تستخدمه

نص يُعلن موقفاً جوهرياً. ليس مقالاً ولا تحليلاً ولا قصة. **إعلان**.

علامات النص المناسب:
- ضربة افتتاحية حادة قبل أي شرح
- موقف لا مفاوضة عليه
- إيقاع بليغ، جمل قصيرة بين طويلات
- ضربة نهاية تستحق أن تُقتبس وحدها
- يُكتب بصيغة "نحن" أو "هذه قناعتي" أو خطاب مباشر

أمثلة من إنتاج خلدون:
- "خذوا فيدرالياتكم..." (هذا النموذج)
- إعلانات الموقف الوطني السوري
- بيانات الرفض/القبول الواضح

---

## بنية القالب

```
hero        → ضربة افتتاحية بحجم شاشة كامل + ختم تاريخ
manifesto   → فقرات النص بإيقاع تنفّس (long-form prose)
beat        → ضربة نهاية بحجم h1
footer      → اسم + تاريخ + هاشتاجات
```

## اللوحة الافتراضية

تصنيف 1 (ثقيل/أرشيفي) — لكن مع لمسة سوريا قوية:
- `--bg` خلفية رمادية ليلية `#14140f`
- `--ink` نص عظمي مُترَّب `#d9d3c6`
- `--syria-deep` للعناوين الكبيرة `#114845`
- `--syria-soft` للضربات والتوقيع `#A1B8A6`

## كيف تستخدمه

1. انسخ `index.html` و `style.css` إلى `output/`
2. استبدل النص داخل `<h1>`, `<article>`, `<p class="beat">`, `<footer>`
3. عدّل `<title>` و `<meta description>`
4. عاين محلياً
5. لا تنشر قبل المراجعة بـ `skills/02-language-and-tone.md` checklist
