# تجربة بناء موقع بصري

الموضوع: الذاكرة السورية والمسؤولية.

أريد موقعاً عربياً بصرياً قصيراً يشرح فكرة أن حفظ الذاكرة ليس عملاً عاطفياً فقط، بل هو جزء من العدالة ومنع تكرار الجريمة.

النبرة:
- وثائقية
- جادة
- إنسانية
- بدون عنف بصري

المطلوب:
- صفحة HTML عربية RTL
- تصميم سينمائي داكن
- بوستر/غلاف بصري
- خطة صور
- برومبتات صور منفصلة لا تظهر داخل الموقع
